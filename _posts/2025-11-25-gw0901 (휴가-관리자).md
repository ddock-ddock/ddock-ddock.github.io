그룹웨어 - 휴가(관리자)에서는 구성원의 휴가를 관리하고, 회사의 휴가 및 복지 정책을 설정하여 모든 구성원에게 적용합니다.
## 설명


### 휴가 부여
휴가를 부여하는 방법은 다음과 같습니다.

### 데모 가이드

1. VC Works에서 **그룹웨어 > 휴가 관리 > 휴가 현황 관리** 메뉴를 차례대로 클릭해주세요.
	- 관리자 계정이 아닌 경우, 휴가 관리 메뉴에는 접근할 수 없습니다. 
2. **[휴가 부여]** 버튼을 클릭해주세요.
	- 구성원에게 휴가를 부여하거나 초기화 할 수 있는 모달이 노출됩니다. 
3. 휴가를 부여받거나, 부여받은 휴가를 수정할 대상자를 선택해주세요.
	- 여러 구성원을 선택해 일괄 부여 및 수정이 가능합니다. 
	- 구성원을 직접 검색해 추가해주세요. 
4. 생성할 휴가의 연도와 휴가 종류를 선택해주세요. 
5. 부여할 일 수를 선택하세요. 
	- 시간으로 생성 버튼을 클릭하면, 시간으로 휴가를 부여할 수 있습니다. 
6. 활성화 된 **[확인]** 버튼을 클릭해주세요.
## 자주 묻는 질문
> 구성원에게 부여된 휴가를 수정하고 싶습니다. 
{: .prompt-tip }
- 구성원에게 부여된 휴가를 수정할 수 있습니다. 
- 휴가 부여 버튼을 클릭 해, 대상자의 모든 휴가를 초기화하고, 새롭게 생성합니다 항목을 선택해주세요. 
- 새롭게 부여할 휴가의 일수를 입력해주세요. 


### 휴가 항목 설정
휴가를 부여하는 방법은 다음과 같습니다.


### 데모 가이드

<div style="position: relative; box-sizing: content-box; max-height: 80vh; max-height: 80svh; width: 100%; aspect-ratio: 2.013845185651353; padding: 40px 0 40px 0;"><iframe src="https://app.supademo.com/embed/cmigy8qysdlspb7b47a260z3c?embed_v=2&utm_source=embed" loading="lazy" title="Add and Configure Vacation Items in TestWorks" allow="clipboard-write" frameborder="0" webkitallowfullscreen="true" mozallowfullscreen="true" allowfullscreen style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"></iframe></div>

1. VC Works에서 **그룹웨어 > 휴가 관리 > 휴가 항목 설정** 메뉴를 차례대로 클릭해주세요.
	- 관리자 계정이 아닌 경우, 휴가 관리 메뉴에는 접근할 수 없습니다. 
2. **[휴가 항목 추가]** 버튼을 클릭해주세요.
3. 새로 생성할 휴가 항목의 이름을 입력해주세요. 
4. 생성할 휴가 항목의 휴가 소진 여부를 선택해주세요. 
	- 소진 선택 시, 부여된 휴가에서 신청한 휴가만큼 휴가를 소진합니다. 
	- 미소진 선택 시, 부여된 휴가를 소진하지 않습니다. 
5. 활성화 된 **[저장]** 버튼을 클릭해주세요.


### 휴일 설정
휴일을 설정하는 방법은 다음과 같습니다.

### 데모 가이드



1. VC Works에서 **그룹웨어 > 휴가 관리 > 휴일 설정** 메뉴를 차례대로 클릭해주세요.
	- 관리자 계정이 아닌 경우, 휴일 설정 메뉴에는 접근할 수 없습니다. 
2. **[휴일 추가]** 버튼을 클릭해주세요.
3. 생성할 휴일의 날짜를 입력해주세요. 
4. 새로 생성할 휴일 항목의 이름을 입력해주세요. 
5. 활성화 된 **[저장]** 버튼을 클릭해주세요.
