딜소싱·미팅 과정에서 투자 검토 중인 회사, 또는 이미 투자 완료된 포트폴리오에 대한 검토·관리 기록을 남기는 기능입니다.

## 설명

- 기록 작성 경로는 두 가지입니다. 상황에 따라 편한 경로를 선택하면 됩니다.
  - **투자/회수 > 회사별 검토 이력**: 투자심의 등록 이전 단계(딜소싱·미팅)부터 자유롭게 회사를 추가해 기록.
  - **포트폴리오 > 포트폴리오 정보**: 이미 투자 완료된 포트폴리오의 상세 화면에서 사후관리성 기록.
- 기록 유형은 두 가지입니다.
  - `검토/관리 기록`: 미팅 요약·실사 노트 등 자유 양식.
  - `주간 보고`: [주간보고](/posts/wr0003/)에 자동 반영되는 형식으로 작성.
- 임시저장·임시보관함 기능을 지원해 장문 작성 중 손실을 방지합니다.

## A. 투자/회수 > 회사별 검토 이력에서 기록하기

투자심의가 아직 등록되지 않은 초기 단계 회사도 이 경로에서 바로 기록을 남길 수 있습니다.

#### 1. 페이지 이동

`투자/회수 > 회사별 검토 이력` 메뉴를 클릭해주세요.

![회사별 검토 이력 진입](assets/img/review_history_01.png)

#### 2. 회사 등록 (이미 등록된 회사는 생략)

1. 우측 상단 **[회사 등록]** 버튼을 클릭해주세요.

    ![회사 등록 버튼](assets/img/review_history_02.png)

2. 필수 정보를 입력한 뒤 우측 상단 **[저장]** 버튼을 클릭해주세요.

    ![회사 정보 입력](assets/img/review_history_03.png)

> 이미 투자심의 또는 포트폴리오가 추가된 회사라면 신규 등록 절차를 생략하고, 목록에서 해당 회사 행을 클릭해주세요.
{: .prompt-tip }

#### 3. 기록 관리 페이지 이동

- 회사를 새로 등록한 경우: 확인 모달에서 **[확인]** 버튼을 클릭해주세요.

![신규 등록 확인 모달](assets/img/review_history_04.png)

- 이미 등록된 회사인 경우: 목록에서 해당 회사의 행을 직접 클릭해주세요.

![회사 행 클릭](assets/img/review_history_05.png)

#### 4. 기록 추가

1. **[+ 기록 추가]** 버튼을 클릭해주세요.

    ![기록 추가 버튼](assets/img/review_history_06.png)

2. 옵션 중 **[검토/관리 기록]**을 클릭해주세요.
	- **[주간 보고]**를 선택하면 해당 기록이 주간보고에 함께 반영됩니다.

![기록 유형 선택](assets/img/review_history_07.png)

3. 제목과 본문을 작성해주세요.

    ![기록 작성](assets/img/review_history_08.png)

4. 작성을 마치면 **[저장]** 버튼을 클릭해주세요.
	- 작성 중 **[임시저장]** 버튼으로 초안을 보관할 수 있으며, 임시 저장한 글은 `임시보관함`에서 불러올 수 있습니다.

![저장](assets/img/review_history_09.png)

## B. 포트폴리오 정보에서 기록하기

이미 투자 완료된 포트폴리오(등기 이후)의 사후관리 기록을 남길 때 사용합니다.

#### 1. 페이지 이동

`포트폴리오 > 포트폴리오 정보` 메뉴를 클릭해주세요.

![포트폴리오 정보 진입](assets/img/review_history_10.png)

#### 2. 포트폴리오 상세 페이지 이동

기록을 남길 포트폴리오를 목록에서 클릭해주세요.

![포트폴리오 행 클릭](assets/img/review_history_11.png)

#### 3. 기록 관리 탭으로 이동

포트폴리오 상세 화면 상단에서 `기록 관리` 탭을 클릭해주세요.

![기록 관리 탭](assets/img/review_history_12.png)

#### 4. 기록 추가

1. **[+ 기록 추가]** 버튼을 클릭해주세요.

    ![기록 추가 버튼](assets/img/review_history_13.png)

2. 옵션 중 **[검토/관리 기록]**을 클릭해주세요.

    ![기록 유형 선택](assets/img/review_history_14.png)

3. 제목과 본문을 작성해주세요.

    ![기록 작성](assets/img/review_history_15.png)

4. 작성 완료 후 **[저장]** 버튼을 클릭해주세요. 임시저장·임시보관함은 A 경로와 동일하게 지원됩니다.

    ![저장](assets/img/review_history_16.png)

## 자주 묻는 질문

> A 경로와 B 경로의 차이는 무엇인가요?
{: .prompt-info }
- A 경로(`회사별 검토 이력`)는 **투자심의가 등록되지 않은 회사**부터 자유롭게 기록을 남길 수 있어 딜소싱·미팅 단계에 적합합니다.
- B 경로(`포트폴리오 정보`)는 **투자 등기가 완료된 회사**의 사후관리 기록을 남기는 데 사용합니다.
- 같은 회사의 기록은 어느 경로로 작성하더라도 한곳에 누적되어 조회됩니다.

> `주간 보고` 유형과 `검토/관리 기록` 유형의 차이는 무엇인가요?
{: .prompt-tip }
- `검토/관리 기록`: 자유 양식. 내부 기록 목적.
- `주간 보고`: 작성된 내용이 [주간보고(wr0003)](/posts/wr0003/) 화면에 자동 반영되어 팀 공유용 주간보고 문서로 활용됩니다.

> 임시저장한 글은 어디에서 불러오나요?
{: .prompt-tip }
- 기록 작성 화면의 `임시보관함` 영역에서 임시 저장된 초안을 조회·편집할 수 있습니다.
