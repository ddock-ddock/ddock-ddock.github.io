회사 정보 (se0051)에서는 시스템 내 출력물에 공통 적용되는 운용사의 대표 정보(기본 정보·연혁·자본금·명판·인감·대표 이메일)를 조회하거나 수정할 수 있습니다.

## 설명

- `설정 > 회사 정보`에서 운용사(법인) 단위의 공통 정보를 관리합니다.
- 회사 정보에 등록된 명판과 인감은 재무제표·운용지시서·조합원 명부·출자증서 등 각종 출력물에 활용됩니다.
- 대표 이메일은 똑똑 주식회사가 VCworks 이용 고객사에 주요 공지 사항(서비스 이용 약관·정책 변경 등)을 발송하는 채널이므로 실제 수신 가능한 주소를 등록해주세요.
- 대표자에 연결할 임직원이 먼저 등록되어 있어야 합니다. 임직원 등록은 [구성원 등록 (hr0001)](/posts/hr0001/) 가이드를 참고해주세요.

#### 회사 기본 정보 수정

1. 우측 상단의 **[수정]** 버튼을 클릭해주세요.

    ![회사 정보 수정 진입](assets/img/se0051_01.png)

2. `기본 정보`, `연혁 정보`, `자본금 변동사항` 영역의 각 항목을 입력해주세요.
3. **[저장]** 버튼을 클릭해주세요.

    ![기본 정보 입력 및 저장](assets/img/se0051_02.png)

> `연혁 정보`와 `자본금 변동사항`은 VICS 정기 보고의 **운용기관 연혁관리**, **운용기관자본금변동관리** 항목에 반영되는 값입니다. LP보고 대상 운용사라면 변경 시 LP보고 영향도를 함께 검토해주세요.
{: .prompt-info }

#### 명판 및 인감 등록

1. 수정 모드에서 업로드할 영역(명판 / 법인인감 / 사용인감)의 **[파일 선택]** 버튼을 클릭해주세요.
2. 이미지 파일을 선택하여 업로드 후 **[저장]** 버튼을 클릭해주세요.
3. `출력물 기본 인감` 설정에서 **법인인감** 또는 **사용인감** 중 출력물에 기본으로 사용할 인감을 선택할 수 있습니다.
	- 이 설정은 운용사 재무제표 출력 시 기본으로 찍히는 인감에 반영됩니다.

![명판·인감 업로드](assets/img/se0051_03.png)

> 업로드한 이미지의 가로·세로 비율이 원본 인감과 다르면 출력물에서 찌그러져 보일 수 있습니다. 배경 투명(PNG) 처리된 정방형 이미지를 권장합니다.
{: .prompt-tip }

#### 대표 이메일 수정

대표 이메일 변경은 인증 코드 확인 절차를 거쳐야 합니다.

1. 회사 정보 화면의 `대표 이메일` 영역에서 녹색 체크 아이콘을 클릭해주세요.

    ![대표 이메일 변경 진입](assets/img/se0051_04.png)

2. 변경할 대표 이메일을 입력한 후 **[인증코드 받기]** 버튼을 클릭해주세요.

    ![인증코드 요청](assets/img/se0051_05.png)

3. 입력한 이메일의 수신함에서 전송된 6자리 인증 코드를 확인해주세요.

    ![인증코드 수신 확인](assets/img/se0051_06.png)

4. 인증 코드를 입력한 후 **[인증 완료]** 버튼을 클릭해주세요.

    ![인증 완료](assets/img/se0051_07.png)

## 자주 묻는 질문

> `설정`에서 `회사 정보` 메뉴가 보이지 않습니다.
{: .prompt-tip }
- 로그인한 ID에 `관리자` 역할이 부여된 경우에만 회사 정보를 조회·수정할 수 있습니다.
- `관리자` 역할이 부여된 계정을 찾기 어려우시면 [we@dkdk.kr](mailto:we@dkdk.kr)로 문의를 남겨주세요.

> 대표 이메일 인증 코드가 오지 않습니다.
{: .prompt-tip }
- 입력한 이메일의 스팸함·정크메일함을 먼저 확인해주세요.
- 사내 메일 시스템에서 외부 발송 메일을 차단하고 있는 경우 인증 메일 수신이 지연되거나 차단될 수 있습니다. IT 담당자에게 `no-reply@ddockddock.net` 도메인의 화이트리스트 등록을 요청해주세요.

> 명판이나 인감을 잘못 등록했는데 어떻게 교체하나요?
{: .prompt-tip }
- 수정 모드에서 해당 영역의 **[파일 선택]** 버튼으로 새 이미지를 업로드하면 기존 이미지가 교체됩니다.
- 이미 출력·배포한 문서의 명판·인감은 자동으로 변경되지 않습니다. 이미 출력된 문서를 재출력하려면 해당 화면(예: 운용지시서·출자증서)에서 재출력 절차를 별도로 진행해야 합니다.

버그 및 문의 사항은 다음 이메일로 보내주세요: **[we@dkdk.kr](mailto:we@dkdk.kr)**
