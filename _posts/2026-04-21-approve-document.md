결재 요청을 받은 구성원이 품의 문서를 검토하고 승인 또는 반려 처리하는 결재자 관점의 절차입니다.

## 설명

- 결재자는 본인이 `결재` 라인에 포함된 문서만 승인·반려할 수 있습니다. `합의`·`처리`·`참조`·`열람` 라인에 포함된 경우 별도 처리 절차가 있습니다.
- 결재 요청은 VCworks 알림·이메일·문서함 세 경로로 확인 가능하며, 어느 경로로 진입해도 같은 문서 상세 페이지로 연결됩니다.
- 품의서 기안자 관점의 등록·요청 절차는 [빈품의서 등록(ed0100)](/posts/ed0100/)과 [계약품의 전자결재(ed0001a)](/posts/ed0001a/)를 참고해주세요.

#### 사전 준비사항

- 본인이 결재 요청 문서의 **결재자 라인**에 포함되어 있어야 합니다.
- 문서 상태가 **결재중**이어야 처리 가능합니다. 이미 승인·반려·회수된 문서는 처리 대상이 아닙니다.

#### 1. 결재 요청 확인

결재 요청을 수신하면 아래 세 경로 중 하나로 문서를 확인할 수 있습니다.

- **문서함**: `전자 결재 > 전체 문서함` 또는 `전자 결재 > 받은 문서(전체 / 결재·합의)`

    ![문서함 진입](assets/img/approve_document_01.png)

- **VCworks 알림**: 홈 화면의 최근 알림 영역에서 결재 요청 건을 클릭

    ![홈 최근 알림](assets/img/approve_document_02.png)

- **이메일**: 결재 요청 이메일의 링크 클릭

    ![결재 요청 이메일](assets/img/approve_document_03.png)

#### 2. 결재 문서 상세 페이지 진입

조회 목록에서 처리할 품의 문서를 클릭해주세요.

![품의 문서 선택](assets/img/approve_document_04.png)

#### 3. 결재 내용 확인

1. 상세 페이지에서 문서 내용과 첨부 자료를 확인해주세요.
2. 우측 상단의 **[반려]** 또는 **[승인]** 버튼을 클릭해주세요.

    ![승인/반려 버튼](assets/img/approve_document_05.png)

#### 4. 결재 처리

1. 팝업에서 승인 또는 반려 사유를 입력해주세요.
    - 승인의 경우에도 의견을 남기는 것을 권장합니다. 결재 이력에 보존됩니다.
2. **[결재승인]** 또는 **[결재반려]** 버튼을 클릭하여 최종 처리해주세요.

    ![사유 입력 및 처리](assets/img/approve_document_06.png)

3. 처리가 완료되면 문서의 결재 상태와 문서 상태가 함께 변경됩니다.

    ![결재 처리 완료](assets/img/approve_document_07.png)

## 자주 묻는 질문

> 다른 사용자가 먼저 결재했는데 제 차례가 오기 전에 반려할 수 있나요?
{: .prompt-tip }
- 결재선이 **순차형**인 경우 이전 결재자가 승인해야 다음 결재자에게 차례가 돌아옵니다. 이전 단계에서 반려되면 본인에게 문서가 도달하지 않습니다.
- 결재선이 **병렬형**인 경우 동시에 진행되므로 본인 차례를 기다릴 필요 없이 바로 처리할 수 있습니다.

> 결재 요청을 수신했는데 알림·이메일이 오지 않습니다.
{: .prompt-tip }
- 알림 수신 설정은 [마이페이지(my0001)](/posts/my0001/)에서 확인하실 수 있습니다.
- 이메일이 스팸함에 들어가 있는지도 함께 확인해주세요.
- `no-reply@ddockddock.net` 도메인이 회사 메일 시스템에서 차단되어 있을 수 있으니 IT 담당자에게 화이트리스트 등록을 요청해주세요.

> 한 번 승인한 결재를 취소할 수 있나요?
{: .prompt-info }
- 현재 승인 완료된 결재의 취소 기능은 제공하지 않습니다. 잘못 승인한 경우 기안자에게 문서 회수를 요청하거나, 후속 품의로 정정 처리를 진행해주세요.

> `합의`·`처리` 라인은 어떻게 처리하나요?
{: .prompt-info }
- `합의` 라인: 승인·반려 권한 없이 의견만 첨부합니다. 본인이 확인한 시점이 결재 이력에 기록됩니다.
- `처리` 라인: 결재 완료 후 실무 처리를 지정받은 담당자입니다. `처리 문서함 > 처리 필요 탭`에서 처리 완료 체크를 진행해주세요.
- `참조`·`열람` 라인: 별도 처리 의무 없이 문서를 확인만 합니다. 확인 여부는 이력에 남습니다.
