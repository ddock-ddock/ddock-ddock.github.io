영업보고서 작성을 위해 투자기업에 대한 심사역 의견을 작성할 수 있는 기능입니다. 검수 완료된 영업보고 내용을 바탕으로 `[AI로 작성]` 초안 생성, `[이전 의견 불러오기]`, 제출본 비교 보기 등을 지원합니다.

## 설명

- 진입 경로: `포트폴리오 > 영업보고 > 심사역 의견 작성`
- 사전 준비사항
    - 투자 스타트업이 STworks에 가입되어 있어야 합니다.
    - 영업보고 제출상태가 `제출 완료` 또는 `재제출` 상태여야 합니다.

### 조회

1. `포트폴리오 > 영업보고 > 심사역 의견 작성` 메뉴로 이동해주세요.

    ![심사역 의견 작성 메뉴 진입](assets/img/br0009_01.png)

2. 좌측 상단 `분기보고` 목록에서 해당 분기를 선택한 후 **[조회]** 버튼을 클릭해주세요.
3. 조회 결과에서 의견을 작성하려는 영업보고 행을 클릭해 상세 페이지로 이동합니다.
    - 기업별 `제출 상태`와 `작성 상태`, 투자 펀드·담당 심사역·최초 투자일을 함께 확인할 수 있습니다.

    ![영업보고 조회 및 행 선택](assets/img/br0009_02.png)

### 의견 작성

상세 페이지로 이동하면 아래 **6개 섹션**이 편집 가능한 상태로 열립니다. 각 섹션에 대한 심사역 의견을 기재해주세요.

| 섹션 | 용도 |
|------|------|
| **투자 포인트** | 투자 결정 당시의 핵심 논리 |
| **사업 진행 현황** | 해당 분기 사업·매출·제품 진행 상황 |
| **사후관리현황** | 보드 참여·지원·모니터링 내역 |
| **EXIT 방안** | M&A, IPO 등 예상 회수 경로 |
| **EXIT 예상 시기/금액** | 시나리오별 예상 시점과 회수 규모 |
| **감액 발생 내역** | 평가 감액 사유 및 조치 |

![6개 섹션 및 AI로 작성 버튼](assets/img/br0009_03.png)

#### AI로 작성

- 각 섹션 입력창 하단의 **[AI로 작성]** 버튼을 클릭하면, **검수된 영업보고 내용**을 토대로 초안이 자동 생성됩니다.
- 생성된 초안은 바로 편집이 가능하므로, 심사역의 관점을 덧붙여 완성할 수 있습니다.

#### 이전 의견 불러오기

- 우측 상단의 **[이전 의견 불러오기]** 버튼을 누르면 **직전 분기에 작성한 의견**이 현재 분기로 복사됩니다.
- 사후관리현황, EXIT 방안 등 분기별 변화가 크지 않은 섹션에서 유용합니다.

#### 제출본 비교 보기

- 우측 상단의 **3줄 아이콘**을 클릭하면 스타트업의 영업보고 제출본이 화면에 함께 표시됩니다.
- 제출본의 수치·내용을 참고하면서 심사역 의견을 작성할 수 있습니다.

#### 저장 및 작성 완료

- **[의견 저장]**: 우측 상단의 버튼을 누르면 현재까지 입력한 내용이 임시 저장됩니다.
- **[작성 완료]**: 작성 상태를 `완료`로 전환합니다. `저장`과 `작성 완료`는 언제든 재전환 가능합니다.

    ![작성 완료 버튼](assets/img/br0009_04.png)

### 투자기업 현황 다운로드

- 우측 상단의 **[투자기업 현황 다운]** 버튼을 클릭하면 영업보고 투자 기업 섹션의 한글 파일이 다운로드됩니다.
- 목록 화면에서 기업을 선택한 후 다운로드하거나, 기업별 작성 화면에서 다운로드할 수 있습니다.

## 자주 묻는 질문

> `[완료]`와 `[저장]`은 어떤 차이가 있나요?
{: .prompt-tip }
- 기능적인 차이점은 없습니다. `[완료]`는 동료들에게 의견 작성이 마무리되었음을 알려주는 상태 표시입니다.
- 작성 완료 이후에도 내용 수정이 가능합니다.

> `[AI로 작성]`은 어떤 데이터를 기반으로 초안을 만드나요?
{: .prompt-tip }
- 해당 분기에 **검수 완료된 스타트업 제출본**을 기반으로 초안을 생성합니다.
- 따라서 영업보고가 `검수 완료` 이전이면 기대만큼의 품질이 나오지 않을 수 있으니, 가급적 검수 완료 후에 사용해주세요.

> `[이전 의견 불러오기]`를 누르면 지금 작성한 내용이 덮어써지나요?
{: .prompt-tip }
- 네, 현재 섹션의 입력값이 직전 분기 의견으로 치환됩니다. 작성 중인 내용이 있다면 **별도로 복사해 둔 뒤** 불러오기를 실행해주세요.

> 심사역 의견은 어디에 출력되나요?
{: .prompt-info }
- 최종 생성되는 **영업보고서 "6. 투자기업 세부현황"** 섹션에 출력됩니다.
- 영업보고서 생성은 [영업보고서 생성(br0011)](/posts/br0011/)을 참조해주세요.

> 제출본 비교 보기 상태에서도 AI로 작성이 가능한가요?
{: .prompt-info }
- 네, 3줄 아이콘으로 제출본을 띄운 상태에서도 각 섹션의 `[AI로 작성]`·`[이전 의견 불러오기]`를 그대로 사용할 수 있습니다.
