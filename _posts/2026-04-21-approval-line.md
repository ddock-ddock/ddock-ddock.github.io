자주 사용하는 결재선 구성을 프리셋으로 저장하고, 이후 품의서 등록 시 한 번에 불러와 재사용하는 기능입니다.

## 설명

- 동일한 구성원에게 반복적으로 결재 요청을 보내는 업무(운용지시서, 품의서, 계약품의 등)에서 결재선 지정 수고를 줄입니다.
- 저장된 결재선은 [빈품의서 등록(ed0100)](/posts/ed0100/) 및 기타 품의서 화면에서 공통으로 불러올 수 있습니다.
- 본인이 저장한 결재선 프리셋만 본인이 불러올 수 있습니다(사용자 단위 프리셋).

## 결재선 저장하기

#### 1. 페이지 이동

`전자결재 > 품의서 등록` 메뉴를 클릭해주세요.

![품의서 등록 진입](assets/img/approval_line_01.png)

#### 2. 결재선 지정

1. 품의서 내용 작성을 마친 후 **[지정]** 버튼을 클릭해주세요.

    ![결재선 지정 버튼](assets/img/approval_line_02.png)

2. 검색 기능으로 결재선에 추가할 구성원을 찾아주세요.

    ![구성원 검색](assets/img/approval_line_03.png)

3. 추가할 라인(결재 / 합의 / 처리 / 참조 / 열람)에 해당하는 버튼을 클릭해주세요.

    ![라인별 추가](assets/img/approval_line_04.png)

> `결재` 라인에 지정된 구성원만 실제 결재 승인·반려 권한을 가집니다. 합의·처리·참조·열람은 확인 권한만 부여됩니다.
{: .prompt-info }

#### 3. 프리셋으로 저장

1. 결재선 구성이 완료된 상태에서 `해당 결재선을 저장합니다` 토글을 **ON**으로 전환해주세요.

    ![토글 ON](assets/img/approval_line_05.png)

2. 저장할 결재선의 명칭을 입력해주세요.

    ![명칭 입력](assets/img/approval_line_06.png)

3. **[결재선 저장]** 버튼을 클릭해주세요.

    ![결재선 저장](assets/img/approval_line_07.png)

## 결재선 불러오기

#### 1. 페이지 이동

`전자결재 > 품의서 등록` 메뉴를 클릭해주세요.

![품의서 등록 진입](assets/img/approval_line_08.png)

#### 2. 저장한 결재선 선택

1. **[불러오기]** 버튼을 클릭해주세요.

    ![불러오기 버튼](assets/img/approval_line_09.png)

2. 저장해 둔 명칭 중 사용할 결재선을 선택해주세요. 선택 즉시 해당 결재선이 품의서에 반영됩니다.

    ![결재선 선택](assets/img/approval_line_10.png)

## 자주 묻는 질문

> 저장한 결재선을 다른 구성원도 사용할 수 있나요?
{: .prompt-tip }
- 결재선 프리셋은 **저장한 본인 계정에만** 속한 개인 프리셋입니다. 다른 구성원이 같은 프리셋을 사용하려면 각자 저장해 두어야 합니다.

> 조직 개편·인사이동으로 결재선 프리셋의 구성원이 퇴사·이동한 경우 어떻게 되나요?
{: .prompt-tip }
- 프리셋에 포함된 구성원이 퇴사하거나 비활성 처리되면 결재선 불러오기 시 해당 인원이 제외되거나 경고가 표시됩니다.
- 조직 개편이 있었다면 주요 결재선 프리셋을 재확인·재저장하는 것을 권장합니다.

> 저장한 결재선 명칭을 수정하거나 삭제할 수 있나요?
{: .prompt-info }
- 현재 프리셋 수정·삭제 UI는 `불러오기` 모달에서 제공합니다. 사용하지 않는 프리셋이 많아지면 목록에서 정리해주세요.
