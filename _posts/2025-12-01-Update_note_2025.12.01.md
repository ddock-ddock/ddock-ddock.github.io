## 투자/회수

### [투자 심의]
#### 업데이트 사항 1. 전체 - 비고 란에서 검토 중단 사유가 긴 경우 전체 확인이 어려운 상태 개선
- 비고 열에 표시되는 검토 중단 사유가 길어지면 ...으로 생략이 되어 전체 확인이 어려웠던 점을 줄 바꿈으로 텍스트가 모두 표시가 되도록 개선했습니다.
- ![12_01_1.png](assets/img/12_01_1.png)

#### 업데이트 사항 2. 신규 거래 유형 : EB (Exchangeable Bond) 의 추가
 - 투자 유형 : 채권의 세부 투자 유형에 교환사채(EB) 거래가 신규 추가 되었습니다.
 - ![12_01_2.png](assets/img/12_01_2.png)


 - ※ 유의 사항 : 현재, 거래 원장 관리에서 사채 교환 거래 등록은 지원되지 않는 상태이며 추후 개발 예정입니다. 개발이 완료되기까지 사채 교환 거래 발생시 별도 요청 주시면 데이터 입력을 도와드리고자 합니다.
### [운용 지시서]
#### 업데이트 사항 3. 재상신 기능 추가
 - 결재가 완료된 운용지시서의 우측 상단 [전자 결재]를 클릭함으로 문서 재 상신이 가능 해졌습니다.
 - 재상신 할 경우 기존 전자결재는 자동으로 취소 처리되며, 취소 문서는 그대로 남아있게 됩니다.
 - ![12_01_3.png](assets/img/12_01_3.png)
### [거래 원장 관리]
#### 업데이트 사항 4. 투자 거래 유형의 상세 페이지 UI/UX 개선
- 거래원장 관리 데이터 수정이 불가능하도록 변경되었습니다.
- 연관된 투자심의 상세 페이지로 이동할 수 있는 링크가 추가되었습니다.
- 인수 방법/시기 섹션의 위치가 변경되었습니다.
#### 업데이트 사항 5. 회수 거래 유형의 상세 페이지 UI/UX 개선
- 거래원장 관리 데이터 수정이 불가능하도록 변경되었습니다.
- 연관된 회수 상세 페이지로 이동할 수 있는 링크가 추가되었습니다.
- 회수 관리에서 첨부된 파일을 확인할 수 있도록 개선되었습니다.

- 회수내역/환산회수내역 필드의 입력 방식이 개선되었습니다.

## 회계/재무
### [법인 카드 사용 내역]

### [전표]
#### 업데이트 사항 11. 전표 증빙 파일이 결재에도 넘어가도록 개선
- 전표 입력 페이지  "증빙"란에 첨부한 파일이 전표 결재에도 파일이 이관되도록 개선 했습니다.
#### 업데이트 사항 12. 전표 입력 UX 개선
- 적요 입력 후 Tab 키를 누르시면 계정코드 첫 번째 행으로 이동되도록 변경되었습니다.
- Tab 키로 첫 번째 행 입력 시 숫자가 지워지던 문제가 수정되었습니다.
- Tab 키로 첫 번째 행 입력 시 한글이 영어로 입력되던 문제가 수정되었습니다.
- 입력된 내용 수정 시 셀이 클릭되지 않던 문제가 해결되었습니다.
- 관리항목 목록에 관리항목을 식별할 수 있는 설명이 추가되었습니다.
- 셀 클릭 시 배경색이 변경되도록 개선되었습니다.
#### 업데이트 사항 13. 계정과목 내 계정코드 입력 형식 변경
- 회계 원장별 계정 과목에서 계정코드에 숫자 외 문자도 입력할 수 있도록 변경되었습니다.

## BI
#### 업데이트 사항 15. 투자심의 단계별 통과 현황을 담당 심사역 별로 통계가 가능 해졌습니다.
- 추가 데이터셋 목록 ▼
	- 1. 회사 단계별 통과현황
		- 회사 기준으로 투자 건에 대한 통과현황 데이터 셋입니다.
	- 심사역별 단계별 통과현황
		- 심사역별 기준으로 투자 건에 대한 통과현황 데이터 셋입니다.
			- ※ 심사역별 기준이기 때문에 한 투자 건에 여러 심사역이 투자한 경우 중복 카운팅이 발생할 수 있습니다.
			- ※ 심사역명으로 필터링하면 해당 심사역의 검토 건수만 확인하실 수 있습니다.

#### 업데이트 사항 18. 조합원 총회 목록 UI/UX 개선
- 총회별 안건에 대한 데이터를 추가적으로 조회할 수 있도록 개선되었습니다.
- ![12_01_6.png](assets/img/12_01_6.png)

#### 업데이트 사항 19. 신규 안건 추가
 - 정기/임시총회 안건 구분에 '재무재표 승인'이 추가되었습니다. (VICS 보고 시 총회 구분은 '기타'로 전송됩니다.)

### [출자/배분]
#### 업데이트 사항 20. 지분양수도 로직 개선
1. 출자 약정액을 양수자에게 이관할 수 있도록 개선되었습니다.
2. 배분이 발생한 조합원의 지분 양도 시 배분 원금, 수익, 배분 조정, 인센티브, 세금(소득세, 법인세, 주민세)을 선택적으로 이관할 수 있도록 개선되었습니다. 
3. 양수도 거래 전 조합원의 기존 출자 배분 내역을 조회할 수 있는 기능이 추가되었습니다.

#### 업데이트 사항 21. 출자/배분 수정 로직 개선
1. 출자/배분과 연결된 운용지시서 및 품의가 있어도 수정이 가능하도록 변경되었습니다.
2. 저장 시 기존 문서는 자동으로 취소 처리 되어 업무 흐름이 더욱 유연해졌습니다.

### [조합원 명부]
#### 업데이트 사항 22. 조합원 상세 정보 UI 개선
1.  '조합원 담당자' UI를 개선하여 동일 담당자를 한 번만 등록하고, 담당 조합을 여러 개 선택해 지정할 수 있도록 변경되었습니다.
2. 중복 입력이 줄고 화면 가독성이 개선되었습니다.

## 설정

### [LPworks 관리]
#### 업데이트 사항 23. LPworks 데이터 전송 시, 비즈니스 분야의 전송 기본값을 사전에 설정할 수 있습니다.
1. 설정 > 사용자 코드 관리 > 비즈니스 분야 설정에 LPworks 코드 연결 항목을 통해 전송 기본값을 설정할 수 있습니다.
2. 데이터를 전송하는 경우, 별도의 추가 입력 작업 없이 LPworks를 사용하는 출자자에게는 연결된 비즈니스 분야로 전송됩니다.
3. 사용자 코드 항목을 추가하는 경우, LPworks 코드 연결 항목을 지정해주셔야 합니다.
4. ![12_01_7.png](assets/img/12_01_7.png)


#### 업데이트 사항 24. 비즈니스 분야를 커스텀한 조합원에게  LPworks 데이터 전송 시, 비즈니스 분야의 전송 기본값을 사전에 설정할 수 있습니다.
1. 설정 > LPworks 관리 > 조합원별 코드 맵핑 탭에서 LP 커스텀 사용자 코드를 연결할 수 있습니다.
2. 데이터를 전송하는 경우, 별도의 추가 입력 작업 없이 LP가 커스텀한 비즈니스 분야로 데이터가 전송됩니다.
3. ![12_01_8.png](assets/img/12_01_8.png)
