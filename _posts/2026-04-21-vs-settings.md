투자 심의 프로세스를 운용사의 업무 방식에 맞게 커스터마이징하는 설정 화면입니다. 단계별 필수값, 심의 단계 구성, 첨부파일 목록 세 가지 영역을 관리합니다.

## 설명

- 진입 경로: `투자/회수 > 설정 > 투자 심의 설정`
- 각 설정이 실제 투자심의 화면에서 어떻게 노출되는지 파악하려면 [투자심의 등록(vs0003)](/posts/vs0003/) 또는 [투자심의 가이드(vs_guide)](/posts/vs_guide/)를 먼저 참고해주세요.

#### 단계별 필수값 설정 (vs0902)

투자심의 단계별로 사용자가 반드시 입력해야 하는 항목의 필수 여부를 제어합니다.

![단계별 필수값 설정 화면](assets/img/vs_settings_01.png)

1. 필수 여부를 변경할 항목의 행을 클릭해주세요.
2. 항목 상세 드로어 우측 상단의 **[수정]** 버튼을 클릭해주세요.
3. `필수 여부` 스위치를 켜거나 꺼서 변경해주세요.
4. **[저장]** 버튼을 눌러 변경 사항을 저장해주세요.

![필수값 수정 드로어](assets/img/vs_settings_02.png)

> 배경색이 회색인 항목은 시스템상 반드시 필요한 정보로 필수 여부를 변경할 수 없습니다.
{: .prompt-info }

#### 단계 설정 (vs0903)

투자심의 단계(미팅 → 예비투자심의 → 본투자심의 → 계약품의 → 납입 → 등기)의 이름을 커스터마이징합니다.

![단계 설정 화면](assets/img/vs_settings_03.png)

1. 커스텀할 단계의 행을 클릭해주세요.
2. 단계 상세 드로어에서 단계명을 변경한 후 **[저장]** 버튼을 클릭해주세요.

> **본투자심의 ~ 등기** 단계는 시스템상 필수 단계로 변경·삭제할 수 없습니다.
> **미팅 ~ 예비투자심의** 단계는 변경 가능하나, 순서는 고정이며 **단계명**만 수정됩니다.
> 배경색이 회색인 항목은 커스터마이징할 수 없습니다.
{: .prompt-info }

#### 첨부파일 목록 설정 (vs0901)

투자심의 상세 화면의 `첨부파일` 탭 구성(업로드 받을 서류 목록)을 정의합니다.

![첨부파일 목록 설정 화면](assets/img/vs_settings_04.png)

1. **[신규 등록]** 버튼을 눌러 새로운 서류 항목을 추가하거나, 기존 행을 클릭해 정보를 변경해주세요.
2. `항목 유형` 선택에 따라 투자심의 상세의 첨부파일 폴더 구성이 아래 예시와 같이 달라집니다.

![항목 유형별 폴더 구성 예시](assets/img/vs_settings_05.png)

## 자주 묻는 질문

> 필수값을 꺼버리면 투자심의에서 해당 항목을 비워도 등기까지 진행되나요?
{: .prompt-tip }
- 네, 필수 여부를 `부`로 변경하면 해당 항목이 비어 있어도 다음 단계로 진행됩니다.
- 다만 회색 배경 항목(시스템 필수)은 설정과 무관하게 항상 입력이 필요합니다.

> 단계 순서를 바꾸거나 단계를 추가·삭제할 수 있나요?
{: .prompt-tip }
- 현재는 **단계명 변경만** 지원합니다. 순서 변경·단계 추가/삭제는 제공하지 않습니다.
- 극단적으로 예비투자심의를 건너뛰고 본투자심의부터 진행하려면, [투자심의 등록(vs0003)](/posts/vs0003/)에서 단계를 직접 점프하는 방식으로 운영할 수 있습니다.

> 첨부파일 항목 유형은 어떻게 선택하면 되나요?
{: .prompt-info }
- `항목 유형`은 폴더 구조를 결정합니다. 예를 들어 "다건 업로드" 유형은 해당 항목에서 여러 서류를 함께 관리할 수 있는 폴더로 생성됩니다.
- 상세 화면의 첨부파일 탭에서 설정대로 폴더가 구성되는지 실제 투자심의 1건으로 확인 후 운영에 반영하는 것을 권장합니다.
