의결 정보 반영(sa0003)에서는 포트폴리오가 회신한 의결 결과 정보를 **거래 원장과 포트폴리오에 자동 반영**하는 방법을 설명합니다. 증자·감자·분할·병합 등으로 인한 발행 주식 수·액면가·지분율 변동을 수동 등록하지 않고 일괄 반영할 수 있습니다.

## 설명

- 진입 경로: `포트폴리오 > 의결`
- **정보 반영이 가능한 의결 안건**
    - 무상증자
    - 주식병합
    - 액면분할
    - 현금배당
    - 유상증자
    - 자본금 증가
    - 자본금 감소
- **사전 준비사항**
    - 대상 의결의 `진행 상태`가 `결과 회신`이어야 합니다.
    - `결과 반영 상태`가 `반영 필요`여야 합니다.
    - 의결의 `마감일시`가 지난 이후여야 실행할 수 있습니다.

### 업무 가이드

1. `포트폴리오 > 의결` 메뉴로 이동한 후, 반영할 의결의 **좌측 체크박스**를 선택해주세요.

    ![대상 의결 선택](assets/img/sa0003_02.png)

2. 상단의 **[결과 정보 반영]** 버튼을 클릭하면 결과 정보반영 화면으로 이동합니다.
3. 의결 결과 정보반영 화면에서 다음 순서로 진행해주세요.
    - 반영 **안건 1건**을 선택합니다. (일괄 처리가 불가하여 안건별 1건씩 실행)
    - 내역 중 **반영 대상**을 체크로 선택합니다.
    - 필수 항목 중 빈 값이 있는 셀을 **더블 클릭**하여 내용을 입력합니다.
    - **[반영하기]** 버튼을 클릭합니다.

    ![결과 정보 반영 선택과 입력](assets/img/sa0003_03.png)

    > 반영 대상 리스트에서 **초록색 블록**으로 표시된 건은 이미 반영이 완료된 내역입니다.
    {: .prompt-info }

4. 확인 팝업이 표시되면 **[확인]** 버튼을 클릭하여 반영을 최종 실행해주세요.

    ![정보 반영 실행 팝업](assets/img/sa0003_04.png)

5. 반영이 완료되면 화면 하단에 `반영되었습니다` 토스트가 표시되고, 안건의 상태가 **반영 완료**로 전환됩니다.

    ![반영 결과](assets/img/sa0003_05.png)

    ![반영 완료 상태](assets/img/sa0003_06.png)

### 반영되는 데이터

- 증자·감자·소각·분할·병합 등으로 인한 **발행 주식 수**, **액면가**, **지분율**의 변동 사항이 [거래 원장 관리(pm0001)](/posts/pm0001/)에 자동 등록됩니다.
- 등록된 내역은 포트폴리오 정보에도 반영되어 이후 영업보고서·LP보고 등에서 활용됩니다.

## 자주 묻는 질문

> `[결과 정보 반영]`을 클릭했는데 빈 테이블이 나옵니다.
{: .prompt-tip }
- 의결 목록 화면에서 `결과 반영 상태`가 `반영 필요`인 의결이 있는지 확인해주세요.
- `결과 반영 상태`가 `반영 필요`인 의결만 결과 반영이 가능합니다.

> 결과 반영으로 인해 생성된 데이터는 어디에서 확인하나요?
{: .prompt-tip }
- `투자/회수 > 거래 원장 관리` 메뉴에서 확인하실 수 있습니다.

> `결과 반영 상태` 값별 의미가 궁금합니다.
{: .prompt-tip }
- `반영 필요`: 정보 반영을 실행해야 하는 의결
- `해당 없음`: 의결 안건이 정보 반영 대상(무상증자·주식병합·액면분할·현금배당·유상증자·자본금 증감) 외인 경우
- `반영 완료`: 이미 정보 반영을 마친 의결

> 안건 여러 개를 한 번에 반영하고 싶은데 `[반영하기]`가 먹히지 않습니다.
{: .prompt-info }
- **안건별로 1건씩**만 반영할 수 있습니다. 안건을 순서대로 선택해 실행해주세요.

> `[반영하기]`를 눌러도 실행이 되지 않습니다.
{: .prompt-info }
- 의결의 `마감일시`가 지나지 않았거나, `진행 상태`가 `결과 회신`이 아닐 가능성이 높습니다. 두 조건이 모두 충족되어야 실행이 가능합니다.
